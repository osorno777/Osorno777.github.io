Admin Center

The PageSearch Admin Center is a very useful tool in which you can administrate all of the features of the PageSearch program.   They include:  Create New Index, Delete Old Index, Edit Header/Footer, Edit Block List.

Admin Center : Loggin in

To create a login and password goto padmin.cgi a login info form will come up, fill in the login information, hit the "Edit" button, click the link on the next page, and login successfully to the admin center from the login page (the link will take you there).

To login into the PageSearch Admin Center after the initial setup, bring up login.html in your browser, enter your password and userid.

Admin Center : Indexing Site (Creating New Index)

There are two ways of indexing your site:  indexing in bits, and indexing as a whole.  It is recommended that you index your site in bits so it will cut down on the process on your web server.  Usually, when you index your site as a whole, it will timeout, which means the index was not created.

To index your site, check the checkboxes next to the files you wish to have in your index.  If you don't want it, don't check it.  To index an entire directory, and all sub-directories (which is not recommended), check the box next to the directory.  To index your site in bits, first select all the files in the current directory that you wish to index, then click on each directory link that you wish to index, and index the files in that directory.  If you index your site in bits, you are basically indexing it directory by directory.

NOTE:  If you ever get a timeout message in your browser, try de-selecting some of the files that you wish to index.

Admin Center : Deleting Old Index

If you ever wish to re-index your entire site, you should delete the old index first.  If you do not, then the old index of your old site will still be there.  Also, if you just want to update the index after you have first indexed your entire site, then you must delete the old index.  The old material from your site will still be available for search if you do not.

NOTE:  You do NOT have to delete the old index each time you are indexing a directory.  The program works where it appends to the index.  You only need to delete the old index if you have updated your site (and wish to re-index your entire site again).

Admin Center : Editing the PageSearch Results Header/Footer

To change the look of the PageSearch Results page, use the Edit Header/Footer button.  This will bring up two textareas which contain the contents of header.html and footer.html.  After editing the html, press the edit button.

Admin Center : Editing the PageSearch Block List

The PageSearch block list is used to block certain file extensions from being indexed.  Such file extensions might contain .cgi, .gif, and .jpg.  Others are supported.  To add a file extension to the block list, enter the file extension (in the format of ".<extension>"), select the Add a new block command from the pull-down menu, and hit the Edit Button.   To delete a block from the list, select the file extension you wish to delete from the pull-down menu, select the Delete a block command, then hit the Edit button.

Search Page

The search page is located at http://<yourdomain.com>/pagesearch/pagesearch.html.  You can change the look as you wish.
